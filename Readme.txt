Explorer Switcher
2015-07-27 by F.Phoenix
***********************

Утилита командной строки для работы с Проводником Windows. Не имеет собственного интерфейса и рассчитана на запуск из других программ: расширителей контектного меню, менеджеров горячих клавиш и прочее.

Параметры запуска:
-sh	переключает отображение скрытых файлов
-se	переключает отображение расширений
-ue	обновляет ассоциации типов файлов (заставляет перечитать из реестра)

-----

Command line tool for Windows Explorer. It has no own GUI and made for using with another programs: context menu extensions, hot key managers etc.

Arguments:
-sh	switch visibility of hidden files
-se	switch visibility of extentions
-ue	update file associations (reload it from registry)
